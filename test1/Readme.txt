In diesem Ordner sind alle Dateien/Ordner für die Demo, die auf dem Hardkernel test1 im Startverzeichnis aufzufinden sind. 
Nachdem Start muss das Skript "bridge.sh" einmal ausgeführt werden, um eine MACsec-Verbindung mit dem Hardkernel Test2 aufzubauen.
Zu beachten ist, dass die MACsec Konfiguration(Verschlüsselungsalgorithmus, Cipherbit, etc..) auf beiden Hardkerneln gleich eingestellt ist.
Sonst kann kein Verbindungsaufbau erreicht werden.
